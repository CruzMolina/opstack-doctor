# opstack-doctor v0.1.11 Release Notes

`v0.1.11` adds shell completion generation.

## Highlights

- Add `opstack-doctor completion bash`.
- Add `opstack-doctor completion zsh`.
- Add `opstack-doctor completion fish`.
- Complete top-level commands, common subcommands, flags, and supported flag values.
- Add completion smoke coverage to local and Docker release checks.

## Operator Impact

- Operators can generate completion scripts from the same binary they install.
- Interactive config validation, checking, demo, export, and generate workflows are easier to discover.
- The implementation remains dependency-free and does not change diagnostic behavior.
